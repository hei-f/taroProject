# Chat App

## 项目简介
这是一个使用 Taro、React、Mobx 和 NutUi 构建的聊天应用，使用openAi Api，需要自行提供ApiKey。

## PC端使用
### 1. 安装项目
```bash
cd taroProject
pnpm install
```
### 2. 编译项目
```bash
cd chat-app
pnpm dev:weapp
```
### 3. 查看项目
前往微信开发者工具查看项目。
